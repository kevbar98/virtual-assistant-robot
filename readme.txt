Instalar Python
1. Descargar e instalar la version 3.10.9 de Python
2. En el proceso de instalacion, procurar hacer la instalacion como administrador y agregar al PATH de variables de entorno la ruta de Python 3.10.9

Instalar complementos
1. Descargar e instalar CMake última versión estable.
2. Descargar e instalar Microsoft C++ Build Tools

Instalar librerias para ejecutar proyecto
1. En el terminal de VSCode o del IDE de su preferencia, ejecutar el comando .\venv\Scripts\activate
2. Cuando en la ruta donde se encuentra aparezca el texto (venv) entonces significa que se ha inicializado correctamente una instancia virtual en Python
3. Ejecutar el comando pip install -r requirements.txt

Inicializar proyecto
1. Ejecutar el comando python main.py
